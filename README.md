# Music Visualizer with p5js

**Notice:**

If you clone this repo you should add your own songs in the "*p5/assets/*" folder in **.mp3** format.

**You should also add your songs in the "*songFiles*" array in order to work.**

```javascript
let songFiles = [{
  url: "filename",
  name: "song name"
}]
```
